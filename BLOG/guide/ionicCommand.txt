=== IONIC ===
$ npm install -g ionic
$ ionic -v

$ ionic state restore (Per installare tutti i plugins cordova del file package.json)

$ ionic start ionic2-tutorial-github sidemenu –v2

$ ionic g page PAGENAME
$ ionic g provider SERVICENAME

$ ionic serve --lab (layout per iOS, Android e Windows Phone)
$ ionic run android

$ ionic platform add/update/remove ios/android
$ ionic platform ls

$ ionic plugin add/remove PLUGINNAME
$ ionic plugin ls

$ ionic build ios/android --release

$ ionic resources 
$ ionic resources --icon
$ ionic resources --splash

=== Android change package ===
$ ionic platform remove android

Modificare il file config.xml

$ ionic platform add android

Il file AndroidManifest.xml e il package è stato modificato

=== ADB ===
$ adb devices (lista dei devices collegati) 

$ adb kill-server
$ adb start-server

=== iOS ===
$ sudo npm install --global --unsafe-perm ios-deploy
$ sudo ionic platform add ios
$ sudo ionic build ios
$ sudo ionic emulate ios

Se non funziona per problemi con i plugins, rimuovere i plugin e reinstallarli.
