=== gradle ===
\Android\sdk\tools\templates\gradle\wrapper NOT FOUND quando viene lanciata l'app sul devices...

estrarre il file templates.zip nella directory
\Android\sdk\tools\

Per l'errore:
ANDROID_HOME=D:\Development\Android\sdk
JAVA_HOME=C:\Program Files\Java\jdk1.8.0_65
Error: Could not find gradle wrapper within Android SDK. Might need to update your Android SDK.
Looked here: D:\Development\Android\sdk\tools\templates\gradle\wrapper

è necessario aggiornare cordova alla versione 7 e lanciare il comando
$ cordova platform update android

=== ionic native ===
Se ci sono problemi con plugins NATIVE ciò dipende dalle versioni:
npm uninstall --save @ionic-native/core
npm install --save @ionic-native/core@latest

La prima volta che si crea un'app è sempre necessario installare ionic-native

Ogni provider di ionic-native deve essere inserito in app.module.ts in "providers"

=== Optimize Ionic 2 App for Production ===
app/main.ts

<code>
	import { platformBrowserDynamic } from '@angular/platform-browser-dynamic';
	import {enableProdMode} from '@angular/core';

	import { AppModule } from './app.module';

	// this is the magic wand
	enableProdMode();

	platformBrowserDynamic().bootstrapModule(AppModule);
</code>

$ ionic build android --prod --release

=== Optimize Angular for Production ===
# these are equivalent
ng build --target=production --environment=prod
ng build --prod --env=prod
ng build --prod
# and so are these
ng build --target=development --environment=dev
ng build --dev --e=dev
ng build --dev
ng build
