=== Note ===
Per JSON API USER:
- Se la connessione non è HTTPS aggiungere il parametro "insecure=cool"
- Per la registrazione utenti, abilitare "Impostazioni > Generali > Iscrizione > Chiunque può iscriversi"

Per abilitare READ/ WRITE su WooCommerce (save, update, delete):
- In the left pane of the WooCommerce Dashboard hover over "WooCommerce" and click on settings
- Click on the API tab
- Click on Keys/Apps and then click on the existing API Key
- Set the permissions to "Read/Write" and save changes. 

Di default WooCommerce legge il numero di post definiti nella configurazione di Wordpress. Per cambiare il numero di elementi da leggere: 
Impostazioni -> Lettura -> ELEMENTI
(/easycomande/wp-admin/options-reading.php)

=== JSON API USER ===
AUTHENTICATION:
http://localhost/alefal.it/PROJECTS/wordpress/api/user/generate_auth_cookie/?username=admin&password=admin&insecure=cool
insecure=cool se non esiste una connessione HTTPS

FB CONNECT:
/api/user/fb_connect/?access_token=CAACEdEose0cBADLKmcHWOZCnW4RGU8emG

REGISTER:
- Recuperare ONCE:
	http://localhost/alefal.it/PROJECTS/wordpress/api/get_nonce/?controller=user&method=register
	{"status":"ok","controller":"user","method":"register","nonce":"3818ac2f0d"}
- Registrate:
	http://localhost/alefal.it/PROJECTS/wordpress/api/user/register/?username=test&email=test@domain.com&nonce=3818ac2f0d&display_name=Test&user_pass=test&notify=no

USER:
- INFO:
	http://localhost/alefal.it/PROJECTS/wordpress/api/user/get_userinfo/?user_id=1
- RETRIEVE:
	http://localhost/alefal.it/PROJECTS/wordpress/api/user/retrieve_password/?user_login=test
	{"status":"ok","msg":"Link for password reset has been emailed to you. Please check your email."}


=== JSON API ===
POSTS:

	http://localhost/alefal.it/PROJECTS/wordpress/?json=get_recent_posts
	http://localhost/alefal.it/PROJECTS/wordpress/api/get_recent_posts/		(With user-friendly permalinks configured)

	http://localhost/alefal.it/PROJECTS/wordpress/?json=get_post&post_id=1

http://localhost/alefal.it/PROJECTS/wordpress/?json=get_category_index

TAGS (solo quelli associati ad un post):
