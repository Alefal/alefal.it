########## INSTALL COMPOSER ##########
https://getcomposer.org/


########## LARAVEL ##########
https://laravel.com/docs/5.4
https://laravel.com/docs/5.4/installation

$ composer global require "laravel/installer"
$ laravel new blog

//Compile and Clear all
Dopo aver modificato il file composer.json bisogna compilare con il comando
$ composer dump-autoload
$ composer update

//New key
$ php artisan key:generate

//Cache
composer dump-autoload
composer clear-cache
php artisan cache:clear
php artisan view:clear


########## AUTH ##########
https://laravel.com/docs/5.4/authentication


########## CRUD ##########
$ php artisan make:controller BookController
$ php artisan make:model Book

in routes/web aggiungere
Route::resource('books','BookController');

$ php artisan route:list


########## MIX ##########
Quando viene modificato il file webpack per includere nuovi js o css bisogna ricompilare lanciando il comando

$ npm run dev | production


########## FILE EXPLORER ##########
https://github.com/jasekz/laradrop
http://laradrop.elegrit.com/file-manager-example

in php.ini abilitare "extension=php_fileinfo.dll"

php artisan storage:link


########## INFO FOR PRODUCTION ##########
-> config/filesystems.php storage directory
    LOCAL   -> 'url' => env('APP_URL').'/storage',                              //local
    PROD    -> 'url' => env('APP_URL').'/costaeventi/storage/app/public',       //production

-> File per LOCAL o PROD
    LOCAL   -> .env         |   public/vendor/jasekz/laradrop/js/laradrop.js 
    PROD    -> .env.PROD    |   public/vendor/jasekz/laradrop/js/laradrop.PROD.js

-> config/database.php
    LOCAL   -> conf for local mysql
    PROD    -> conf for remote mysql
