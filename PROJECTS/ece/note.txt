=== Note ===
Per JSON API USER:
- Se la connessione non è HTTPS aggiungere il parametro "insecure=cool"
- Per la registrazione utenti, abilitare "Impostazioni > Generali > Iscrizione > Chiunque può iscriversi"

Per abilitare READ/ WRITE su WooCommerce (save, update, delete):
- In the left pane of the WooCommerce Dashboard hover over "WooCommerce" and click on settings
- Click on the API tab
- Click on Keys/Apps and then click on the existing API Key
- Set the permissions to "Read/Write" and save changes.

=== BACKOFFICE WHE (WhareHouseEngine) ===
Sistema basato su Wordpress con i seguenti plugin:
- JSON API
	chiamate JSON per recuperare i posts inseriti
- JSON API User
	chiamate JSON per accesso al sistema
- User Role Editor
	gestione ruoli aggiuntivi (fornitori | clienti)
- WooCommerce
	ecommerce con PRODOTTI, ORDINI, ecc.
- WooCommerce - Store Exporter Deluxe
	ext di woocommerce per l'export in CSV, XML, XLS, XLSX
- WooCommerce Print Invoice & Delivery Note
	ext di woocommerce per la stampa della fattura
- WooCommerce Stock Manager
	ext di woocommerce per la gestione veloce del magazzio
- alefal_woocommerce
	ext di woocommerce per chiamatea JSON al commerce
